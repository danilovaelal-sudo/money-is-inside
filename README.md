# Деньги внутри

Интерактивное веб-приложение на `Next.js`, созданное по книге и рабочей тетради «Деньги внутри».

Это не финансовый советник и не курс про быстрые деньги. Приложение помогает пройти 15-дневный личный маршрут через вопросы, практики, дневниковые записи и итоговую карту наблюдений.

Репозиторий:
`https://github.com/danilovaelal-sudo/money-is-inside`

## Что внутри

- главная страница с визуальным входом в маршрут;
- онбординг с выбором запроса и режима прохождения;
- маршрут из 15 практических дней;
- отдельная страница для каждого дня;
- личный дневник с ответами по дням;
- итоговая карта «Моя карта денег внутри»;
- страница книги;
- финальный экран;
- настройки и сброс прогресса.

## Стек

- `Next.js` App Router
- `React`
- `TypeScript`
- `Tailwind CSS`
- `Framer Motion`
- `localStorage`

## Как работает сохранение

В первой версии приложение:

- без авторизации;
- без базы данных;
- без оплаты;
- без подписки;
- без ИИ-чата.

Все ответы и прогресс сохраняются локально в браузере через `localStorage`.

Ключ хранения:

`money-inside-progress`

## Страницы

- `/` — главная
- `/start` — онбординг
- `/path` — маршрут из 15 дней
- `/day/[id]` — страница практики
- `/journal` — личный дневник
- `/map` — карта денег внутри
- `/book` — страница книги
- `/final` — финальный экран
- `/settings` — настройки

## Структура проекта


## Быстрый старт

Установить зависимости:

```bash
npm install
```

Запустить локально:

```bash
npm run dev
```

Открыть в браузере:

`http://localhost:3000`

## Полезные команды

```bash
npm run dev
npm run build
npm run start
npm run typecheck
```

## Важно

Ссылка на книгу вынесена в:

`src/lib/constants.ts`

Сейчас там используется заглушка:

`PASTE_LITRES_URL_HERE`

Чтобы кнопка покупки вела на реальную страницу, нужно подставить актуальный URL Литрес.
